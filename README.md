# Python3.WebAPP.Blog
使用Python3开发一个个人Blog网站，包含日志、用户和评论3大部分，可后台管理的个人博客。
基本目录结构如下：

blog-python3-webapp/     <-- 根目录
|
+- backup/               <-- 备份目录
|
+- conf/                 <-- 配置文件
|
+- dist/                 <-- 打包目录
|
+- www/                  <-- Web目录，存放.py文件
|  |
|  +- static/            <-- 存放静态文件
|  |
|  +- templates/         <-- 存放模板文件
|
+- ios/                  <-- 存放iOS App工程
|
+- LICENSE               <-- 代码LICENSE
